## Utilities with Complete Extracted Data

### 1. Lewis County Public Utilities (Washington State)

**Portal**: https://www.lcpud.org/services/engineering-construction/net-metering/

**Application Types**: Net Metering (solar, wind, hydro, fuel cells, biogas)

**System Size**: Up to 100kW

**Fees**: 
- $100-$500 (system dependent, non-refundable)
- $200 meter installation fee

**Key Requirement**: IEEE 1547 or UL 1741 certification

**Insurance**: None required for qualifying systems (RCW 80.60)

**Required Documents**:
- One-line diagram (energy source, disconnect, inverter(s), meter)
- Site map (showing service panel connection)
- Specification sheets (panels, inverters, equipment)
- IEEE 1547 or UL 1741 certification

**Submission Method**: Email (netmetering@lcpud.org) or Mail (PO Box 330, Chehalis, WA 98532)

**Timeline**: ~8-12 weeks typical

**Contact**: netmetering@lcpud.org | (360) 748-9261

---

### 2. Hancock-Wood Electric Cooperative (Ohio)

**Portal**: https://www.hwe.coop/solar-renewable-energy-applicationcontracts-residential

**Application Form**: Simplified Solar Application (2023)

**Fees**: $200 application fee (check payment)

**Key Requirement**: Personal Liability Insurance ($250,000 minimum)

**Required Documents**:
- One-line diagram
- Site drawing (inverter, modules, meter, disconnect switch location)
- Inverter specification sheet
- PV module specification sheet
- Proof of Personal Liability Insurance

**Submission**:
- Email: billing@hwe.coop
- Mail Check: 1399 Business Park Drive South, North Baltimore, OH 45872

**Key Contact**: Caitlyn (ask for on first contact)

## How to Use This for Ambia Enhancements

**For Adding New Utility Profiles**:
1. Open the training guide link
2. Extract: Portal URL, fees, required documents, submission method, contact info
3. Add to `SKILL.md` under "Utility-Specific Profiles" section
4. Update this reference document

**For Phase 1 Context Gathering**:
- Use these guides to know which documents utility requires
- Set fee expectations before gathering
- Know submission timeline

**For Portal Navigation**:
- Reference guide shows portal URLs
- Guides show what documents are "musts" vs optional
