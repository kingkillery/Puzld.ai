## Example Tasks

### Task 1: "Log an approval for Lewis County PUD"
Claude will:
1. Know to look for "Receive and Process IX Part 1" task
2. Create "Interconnection: Approved" task
3. Know format: [MM-DD] – Part 1 Approval received.

### Task 2: "What documents do we need?"
Claude will:
1. Reference the correct utility profile
2. List exact documents with specs
3. Know if insurance is required
4. Provide fee structure

### Task 3: "Find the interconnection task"
Claude will:
1. Search Salesforce by address
2. Navigate to correct project
3. Show all IX tasks with status
4. Reference training guide if needed
